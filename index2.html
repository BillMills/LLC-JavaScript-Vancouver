Functions and Pizza

Let's order a pizza!  We'll create a function called orderPizza that will represent all the steps it takes to place an order (find the pizza place's phone number, call them, give them your address, etc).  Inside the function is where the pizza place will hold our options like toppings, crust type and size of pizza, so that they can save that information and make our order!

function orderPizza() {
  var pizzaOptions = "";
  pizzaOptions += "cheese,";
  pizzaOptions += " thin crust,";
  pizzaOptions += " medium";
  document.write(pizzaOptions);
}
orderPizza();

What if we want to order another pizza but with different options?  Let's order another pizza.

function orderAnotherPizza() {
  var pizzaOptions = "";
  pizzaOptions += "pepperoni,";
  pizzaOptions += " stuffed crust,";
  pizzaOptions += " small";
  document.write(pizzaOptions);
}
orderAnotherPizza();

Global Variables

Let's say we want cheese to always be included as an option in our makePizza function.

var baseTopping = "cheese";
function makePizza() {
 var pizzaOptions = "";
 pizzaOptions += baseTopping + ",";
 pizzaOptions += " bacon,";
 pizzaOptions += " thin crust,";
 pizzaOptions += " medium";
 document.write(pizzaOptions);
}
makePizza();

This is an example of a global variable, since it is declared outside of the function and is accessible from anywhere in the program.
Global Variables vs Local Variables
What happens if we use the same baseTopping variable inside the function?

var baseTopping = "cheese";
function makePizza() {
 var baseTopping = "tomato sauce";
 var pizzaOptions = "";
 pizzaOptions += baseTopping + ",";
 pizzaOptions += " bacon,";
 pizzaOptions += " thin crust,";
 pizzaOptions += " medium";
 document.write(pizzaOptions);
}
makePizza();

What happened to the cheese!?
A local variable is declared inside the function and can only be used within that function.

document.write(baseTopping); //will return “cheese”

Functions, Pizza and Parameters

This could get cumbersome if we wanted to make a lot of pizzas.  If the steps required to make a pizza are the same and only the options change, how can we re-use the makePizza function instead of creating multiple functions?  We use PARAMETERS.  

Functions can be used with one parameter, more than one parameter or none at all.

Live code: start with one parameter and work up to three

function makePizza(topping, crust, size) {
  var pizzaOptions = "";
  pizzaOptions += topping + ",";
  pizzaOptions += " " + crust + ",";
  pizzaOptions += " " + size;
  document.write(pizzaOptions);
}
makePizza("pepperoni", "thick", "large");
makePizza("cheese", "thin", "small");

Arrays

An array is a special variable, it can hold more than one value at a time in a single variable.
If a variable is like a box, then an array is a box with slots for packing wine glasses.
In JavaScript words, arrays store an ordered list of values in slots, in an “indexed” way.

Declare Arrays (yes, capital A) using the NEW keyword.
var robots = new Array();

You can set values using a square bracket [ ] notation.

var robots = new Array();
robots[0] = "Wall-e"; //value for the 1st slot in the array
robots[1] = "Eva"; //value for the 2nd slot in the array
robots[2] = "The tin man"; //value for the 3rd slot in the array

The numbers inside the square brackets [ ] are referred to as the index number.
Array indexes always start at zero.
Arrays and Indexes
The index numbers are also used to access the value of the specific slot in the array.
To access the value of the first robot in the robot array, index 0 is the first item in the list.

var robots = new Array();
robots[0] = "Wall-e";
robots[1] = "Eva";
robots[2] = "The tin man";
document.write("The 2nd robot is " + robots[1] + " even though the index is 1" + ".");

Note: There are other ways to declare an array but we will stick to this syntax for the remainder of the workshop.  
More info here: http://www.w3schools.com/js/js_obj_array.asp

Arrays & Length and the 0 Index


How do we access a particular value?
By using the index number.

var robots = new Array();
robots[0] = "Wall-e";
robots[1] = "Eva";
robots[2] = "The tin man";

So, to access the last index value, you can use robots[2].
BUT...
You can also use robots[ robots.length-1 ].

Huh?

robots.length would return a value of 3 because there are 3 elements in the array.
But the index of the last element is actually [2] because the array starts at [0].
robots.length = 3
index of last element = 2
robots[2] = robots[robots.length-1]

Since the index begins at 0, the last index value is always length-1.
This can be useful if you want to always access the last index despite new additions.

Loopty-Loops

Loops are used to execute the same code continuously using conditional statements to test when the code will begin and stop.

Why use loops?  Because sometimes we need to execute repetitive code through a list of values or for a specified number of times.

It's like creating a playlist on your favorite music player.  
You create a list of songs, press play and then each song is played automatically in the order set out in the playlist.  
When it reaches the end of the playlist, it stops.

We will be discussing while loops and for loops.

While Loop


The while loop executes the block of code contained in the loop while the condition is true.   
An index variable is required to keep track of its place in the loop.  
The while loop will stop executing the code once the condition becomes false.

If you had 10 songs in your playlist, the player won't stop until it reaches the end of song 10.

//declare index variable
var i = 0;
while ( i < playlist.length ) {
 // play current song represented by var i
 i++;
 // i++ will add 1 to var i so it moves to the next song
}


NOTE:  If the condition is always true, the loop will run infinitely and will never exit the while loop.  
This can cause your browser to crash!

For Loops


A for loop loops through a block of code a specified number of times.  
The index, conditional test and incrementer are all declared in one line.
The index (usually declared as var i) is used as a counter and defines the start value, followed by the termination condition and an increment value.  

// start value, condition, increment
for ( var i = 0; i < 3; i++ ) {
 //code to be executed
 document.write("The current index is: " + i + "<br />");
}
document.write("The for loop is now complete.");

For Loops and Arrays

For loops are useful when used it conjunction with arrays because it can be used to execute specific actions while going through an array list of values.  

Back to our robot store.  It wouldn't be much of a store if there was only one product!
